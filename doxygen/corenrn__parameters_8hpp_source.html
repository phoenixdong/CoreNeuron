17 /**
18  * \class corenrn_parameters
19  * \brief Parses and contains Command Line parameters for Core Neuron
20  *
21  * This structure contains all the parameters that CoreNeuron fetches
22  * from the Command Line. It uses the CLI11 libraries to parse these parameters
23  * and saves them in an internal public structure. Each parameter can be
24  * accessed or written freely. By default the constructor instantiates a
25  * CLI11 object and initializes it for CoreNeuron use.
26  * This object is freely accessible from any point of the program.
27  * An ostream method is also provided to print out all the parameters that
28  * CLI11 parse.
29  * Please keep in mind that, due to the nature of the subcommands in CLI11,
30  * the command line parameters for subcategories NEED to be come before the relative
31  * parameter. e.g. --mpi --gpu gpu --nwarp
32  * Also single dash long options are not supported anymore (-mpi -> --mpi).
33  */
